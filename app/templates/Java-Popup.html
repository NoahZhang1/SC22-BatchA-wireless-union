The Project

Model Creation Procedure

We created 6 separate models, one for each respective programming language(C, C++, C#, Python, Java, and JavaScript), using various models to do so.


  • GPT-2-DoubleHead (C, Python, JavaScript, Java)
  • DistilGPT (C#)
  • GPT-Neo-125M (C++)

    • Recieves and analyses user input

    This project has the ability to take user inputs and use them as prompts to give code based on the algorithm you request. These models were trained on top-of-the notch algorithms, including Binary Search algorithms, Bubble Sort, and Palindrome analysis.




    Saves time during programming

    The ability to reference an AI is an extraordinarily beneficial thing with respect to speed of programming, and referencing this and other AIs can bring about new perspectives to the problem that you could not have come up by yourself.


    All Outputs are Completely Unique

    The AIs we created are designed to generate that have never existed before, bypassing the repeated annoyance of tutorials that state the exact same thing on repeat. By creating unique pieces of code, it allows for a perfectly new perspective to be found in the form of code.
